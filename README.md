## 📜 Descrição
Projeto desenvolvido como desafio técnico do programa **Oracle Next Education (ONE)** em parceria com a **Alura**. Trata-se de uma aplicação web interativa capaz de codificar e decodificar textos, servindo como uma ferramenta prática para o exercício de lógica de programação e manipulação do DOM.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5:** Estruturação semântica e acessível.
- **CSS3:** Estilização responsiva e design de interface.
- **JavaScript:** Lógica central de substituição de caracteres e manipulação dinâmica de elementos.

---

## ⚙️ Funcionalidades
- **Codificação:** Substitui vogais específicas por padrões definidos para ocultar o texto original.
- **Decodificação:** Processo reverso que detecta o padrão e restaura a mensagem.
- **Validação:** Implementação de RegEx para restringir a entrada apenas a letras minúsculas sem acentuação.
- **Área de Transferência:** Botão funcional para copiar o resultado gerado.
